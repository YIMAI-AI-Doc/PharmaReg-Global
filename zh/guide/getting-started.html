快速开始 ​

欢迎使用PharmaReg-Global！这是一个类似 VitePress 官网的静态文档站。

前置准备 ​

确保你已安装 Node.js（推荐 18.x 或 20.x）。

初始化项目 ​

bash
npm init -y
npm install -D vitepress
